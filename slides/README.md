# Markdown Slide Decks

Presentation-ready **markdown** slide files for facilitators. No PowerPoint required — project these in any markdown viewer, VS Code preview, Marp, or print as handouts.

---

## Format

Each file uses this pattern:

```markdown
# Slide NN | Title

## On screen
(bullets, tables, diagrams — what the room sees)

## Speaker notes
(what to say, timing, activity cues)

---
```

- `---` separates slides  
- **Lecture doc** = full content; **slides file** = delivery view  
- Activities: slide says “START ACTIVITY” — details in lecture doc

## Tips for facilitators

1. Read **speaker notes** — they are not shown on screen  
2. Pause on **diagram slides** — let the room read  
3. Run **activity slides** from the linked lecture doc timing  
4. Keep lecture doc open for Q&A prepared answers
