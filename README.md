## Suggested paths

**General literacy (non-technical focus):** 01 → 03 → 02 → trust/privacy sessions in 01 series plan

**Technical literacy (GenAI stack):** 03 (recap) → **04 → 05 → 06** (~4–4.5 hours total)

Each of Lectures 4–6 is designed for **90–110 minutes** including discussion and Q&A.

**Lecture 4** adds: neural networks, training math intuition, Transformer structure, tokens & embeddings.  
**Lecture 5** adds: hardware VRAM/RAM tables, software stack specs, deployment tiers.  
**Lecture 6** adds: vectorization, embedding models, distance metrics, ANN indexes.
