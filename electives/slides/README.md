# Elective Slide Decks

Presentation-ready **markdown** slide files for ST-EOS-REF electives. Project in any markdown viewer, VS Code preview, Marp, or print as handouts.

---

## Format

Each file uses this pattern:

```markdown
# Slide Deck — Title

*Source: [lecture.md](../lecture.md)*

---

# Slide NN | Title

## On screen
(bullets, tables, diagrams — what the room sees)

## Speaker notes
(2–4 sentences — what to say, timing, activity cues)

---
```

- `---` separates slides  
- **Elective doc** = full content; **slides file** = delivery view  
- Activities: slide says **START ACTIVITY** — details in the linked elective doc

**Core curriculum slides:** [../../slides/](../../slides/)

## Tips for facilitators

1. Read **speaker notes** — they are not shown on screen  
2. Pause on **diagram and table slides** — let the room read  
3. Run **activity slides** using timing in the elective doc  
4. Keep the elective doc open for Q&A and edge cases  
5. **E04** and **E06** — encourage leadership attendance  
6. **E06** — add your jurisdiction's official AI guidance verbally; slides are not legal advice
