# Elective E02 — AI You Already Use Every Day

*Short elective · 45–60 minutes · All members*

**Slides:** [slides/E02_ai_daily_life.md](slides/E02_ai_daily_life.md)

---

## Objectives

1. Name **Recognition AI** in daily tools  
2. Name **Generative AI** now visible in consumer apps  
3. Build intuition before org platform use  

---

## 1. Opening icebreaker (5 min)

List AI you used today: phone, maps, search, translation, social feed, chat assistant.

---

## 2. Recognition AI — invisible layer (15 min)

| Tool | AI job |
|------|--------|
| Email spam filter | Classify spam |
| Face unlock | Match face |
| Voice assistant wake word | Detect phrase |
| Maps traffic | Predict delay |
| Streaming recommendations | Rank content |
| OCR on phone camera | Recognize text |

**Pattern:** Mostly **behind the scenes** — you see outcome, not prose.

---

## 3. Generative AI — visible layer (12 min)

| Tool | AI job |
|------|--------|
| Chat assistants | Draft / answer |
| Translation (modern) | Generate target text |
| Image generators | Create pixels |
| Meeting summaries | Draft notes |

**Pattern:** You **read** the output — verify before trust ([08_trust_verification.md](../08_trust_verification.md)).

---

## 4. Stack diagram — both together

```
Photo of document
    → OCR (recognition)
    → RAG search (retrieval)
    → LLM summary (generative)
    → You verify (human)
```

---

## 5. Activity — classify the feature (10 min)

| Feature | R, G, or Both? |
|---------|----------------|
| Autocomplete in search box | Both |
| Fraud alert on card | R |
| Chatbot draft reply | G |
| Sort inbox priority | R |

---

## 6. Work vs personal (5 min)

Org may **approve** internal stack ([13_post_launch_platform.md](../13_post_launch_platform.md)) — personal apps may not meet data rules ([09_privacy_security.md](../09_privacy_security.md)).

---

## 7. Takeaways

1. You already use AI daily — mostly recognition.  
2. Generative AI is newer **visibility** — draft, don’t assume truth.  
3. Org assistants combine both + RAG + you.
